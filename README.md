# Vagrantfile

## Dependencies

* [Oracle VritualBox](https://chocolatey.org/packages/virtualbox)
* [Vagrant](https://chocolatey.org/packages/vagrant)

## Installation

```bash
git clone http://github.com/ndxbn/vagrantfile
cd vagrantfile
./install.sh
vagrant up
```

If your platform is Windows, use `install.bat` .
